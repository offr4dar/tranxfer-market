# Tranxfer Market — Documentation

Welcome to the **Tranxfer Market** developer and product documentation.

Tranxfer Market is a React Native mobile marketplace for football transfers. Players, agents, and clubs can discover each other, send connection requests, and manage conversations — all behind an authenticated, role-based onboarding flow.

---

## What is this?

Tranxfer Market is built to solve a real problem in grassroots and semi-professional football: **visibility**. Players struggle to be discovered. Agents have no centralised tool to manage shortlists. Clubs spend too much time sourcing talent through informal networks.

This platform changes that.
