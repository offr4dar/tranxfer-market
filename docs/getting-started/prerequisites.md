# Prerequisites

Before running Tranxfer Market locally, make sure you have the following installed and configured.

## Required Tools

| Tool | Version | Notes |
|---|---|---|
| Node.js | ≥ 18 | LTS recommended |
| npm | ≥ 9 | Comes with Node |
| Expo CLI | Latest | `npm install -g expo-cli` |
| Git | Any | For version control |
| Xcode | ≥ 15 | Required for iOS Simulator (macOS only) |
| Android Studio | Latest | Required for Android Emulator (optional) |

## Required Accounts

- **[Clerk](https://clerk.com)** — Create a free account and a new application. You will need the publishable key.
- **[Supabase](https://supabase.com)** — Create a free project. You will need the project URL and anon key.
- **[Expo Account](https://expo.dev)** — Required for running on a physical device via Expo Go.

## Physical Device (Optional)

Install **[Expo Go](https://expo.dev/go)** on your iOS or Android device to preview the app without building natively.
